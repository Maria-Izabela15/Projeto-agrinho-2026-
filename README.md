# 🌱 Fazenda do Futuro | Agro Inteligente

Uma landing page que apresenta o futuro da agricultura sustentável no Brasil. O projeto une tecnologia de precisão com um design moderno e interativo, demonstrando que produtividade e preservação ambiental podem caminhar juntas.

---

## 🎯 Objetivo do Projeto

Apresentar o modelo de fazenda do futuro, showcasing tecnologias como IoT, Drones com IA e Satélites, de forma moderna e interativa, com animações e transições suaves.

---

## 🛠️ Recursos Técnicos

### ✨ Interatividades (JavaScript)

- **Menu Mobile:** Toggle responsivo com animação, fecha ao clicar fora ou em links
- **Typing Effect:** Texto digitado animadamente no título principal
- **Cards Interativos:** Revelação de dados extras no hover e foco
- **Smooth Scroll:** Navegação suave com offset para navbar fixa
- **Navbar com Scroll:** Transição de estilo ao rolar a página
- **Scroll Reveal:** Animação de entrada dos elementos
- **Citações Rotativas:** Alternância automática de citações
- **Back to Top:** Botão flutuante para retornar ao topo
- **Polyfill:** Suporte a scroll-behavior em navegadores antigos

### 🎨 UI/UX & Design

- **Design Responsivo:** Totalmente adaptável para móveis e desktop
- **Glassmorphism:** Efeito de vidro na navbar e cards
- **Animações CSS:** Transições suaves em todos os elementos
- **Tipografia:** Hierarquia clara com Google Fonts (Poppins)
- **Ícones:** Font Awesome para ícones visuais

### 💻 Tecnologias Utilizadas

- **Estrutura:** HTML5 Semântico
- **Estilização:** CSS3 (Variáveis CSS, Flexbox, Grid)
- **Tipografia:** Google Fonts (Poppins)
- **Ícones:** Font Awesome 6
- **Interatividade:** JavaScript Vanilla (ES6+)
- **Imagens:** Unsplash

## 🚀 Funcionalidades

### ✅ Menu Mobile
- Toggle responsivo com ícone animado
- Fecha ao clicar em qualquer link
- Fecha ao clicar fora da área do menu
- Bloqueia scroll da página quando aberto

### ✅ Typing Effect
- Texto rotativo: *"Futuro Sustentável"*, *"Agro Inteligente"*, *"Tecnologia Verde"*, *"Biomas Protegidos"*
- Velocidade adaptativa (100ms escrita, 50ms deleção, 2s pausa)

### ✅ Cards Interativos
- 3 cards: Sensores IoT, Drones com IA, Satélites
- Revelação de dados extras no hover
- Suporte a foco via teclado (`tabindex="0"`)
- Atributos ARIA para acessibilidade

### ✅ Scroll Suave
- Offset automático para navbar fixa
- Suporte a todos os links internos (`#anchor`)

### ✅ Navbar com Scroll
- Estilo muda após 100px de scroll
- Efeito glassmorphism com `backdrop-filter`

### ✅ Scroll Reveal
- Elementos aparecem ao entrar na viewport
- Animação de `opacity` e `translateY`

### ✅ Citações Rotativas
- 4 citações sobre sustentabilidade
- Alternância a cada 8 segundos
- Transição suave de `opacity`

### ✅ Back to Top
- Botão aparece após 300px de scroll
- Animação de `opacity`
- Retorno suave ao topo
